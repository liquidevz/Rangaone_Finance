# Sounds Directory

This directory contains audio files used throughout the application.

## Required Files

### cart-add.mp3
A short, pleasant sound effect that plays when users add items to their cart.

**Specifications:**
- Format: MP3
- Duration: 0.5-1.5 seconds
- Volume: Should be pleasant and not jarring
- Suggested sound: A soft "ding" or "chime" sound

**Where to find suitable sounds:**
- [Freesound.org](https://freesound.org) (free with attribution)
- [Zapsplat](https://zapsplat.com) (free with account)
- [Mixkit](https://mixkit.co/free-sound-effects/) (free)

**Example search terms:**
- "cart add"
- "notification chime"
- "success ding"
- "button click"
- "positive notification"

## Usage

The cart sound is used in:
- `components/model-portfolio-section.tsx` - When adding portfolios to cart

The application will work without the sound file, but users won't hear audio feedback.
